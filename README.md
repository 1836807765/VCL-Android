# VCL-Android

VLC很强大，支持字幕，支持锁定屏幕，支持跳转到某个具体时间，支持0.25-4倍的慢放、快放、支持亮度调节、支持音量调节、支持快进、支持直播等特性，是做在线视频播放的绝佳选择之一！！！！

这是编译好的VCL-Android版本，打了四种架构的so包，armeabi-v7a、armeabi、x86、mips
